jprotobuf-rpc-common
====================

Common base library for RPC implementation usage.

## 使用示例 ##

```xml
<dependency>
  <groupId>com.baidu</groupId>
  <artifactId>jprotobuf-rpc-common</artifactId>
  <version>1.1</version>
</dependency>
```
